ThinkFix is a **full-stack EdTech web application** designed to help students (JEE/NEET) identify conceptual mistakes in their answers and help coaches analyze class-level learning gaps.

This project is built as part of a learning / internship-ready full-stack system using **Next.js, Node.js, Express, and MongoDB**.
